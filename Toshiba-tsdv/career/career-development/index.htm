Career Development

Capability Model

Toshiba Software Development (Vietnam) have set up an engineering capability model for software engineer. This model is base of engineering activity, especially in improving capability of each engineer. This figure shows overview of engineering capability model. Engineering capability model consist of career path and Talent Model.

Relation between career path and talent model

Career path is job career, and it specifies position of each engineer. Node of career path defines what each engineer has to do in the Company.

In TSDV, we understand that continuous learning/training is one of most important activities to assure sustainable development of any organization. To improve a member’s skills along his/her career path in TSDV, different trainings will be provided in different career levels. There are 03 types of training:

Fundamental skill trainings are carried out in TSDV.

Common skill trainings are carried out by outside training in Vietnam.

Advanced skill trainings are carried out by Toshiba group in Japan and other countries.

Chances To Study And Work In Japan

Since TSDV is developing software for Toshiba group's products & solutions, there are several chances to study & work in Japan for TSDV members. There are 03 main schema including: (i) OJT, (ii) Bridge Engineer, (iii) Long stay, with different objectives & durations. The ultimate purpose are (i) to gather domain knowledge of Toshiba products & solutions and (ii) to skill-up members in both development technologies & professional working style. Proficiency in Japanese language would be great advantage for you to be selected, but it is not the only condition.
